# Следующий клиент

Лендинг четырёхнедельной совместной работы для опытных специалистов, развивающих личную практику, услуги или продукты через тёплый нетворк.

## Project Card

- Objective: помочь посетителю узнать свою ситуацию, поверить в метод и начать разговор о совместной работе.
- Parent/context: самостоятельный активный проект в workspace `ai input`.
- Scope: контент, дизайн, адаптивная страница, CTA, Git и публикация через Sites.
- Out of scope: CRM, платёжная форма, холодные рассылки, рекламные кабинеты и выдуманные кейсы.
- Inputs: исходный текст оффера и подтверждённые пользователем кейсы.
- Outputs: исходники сайта и публичная версия.
- Current stage: первая версия собрана и готовится к публикации.
- Definition of done: production build проходит, версия опубликована, CTA работает.
- Next action: указать прямую ссылку на Telegram/Calendly, когда она будет известна.
